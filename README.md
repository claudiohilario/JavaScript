# JavaScript - Apontamentos
## Operadores Matemáticos
| Operador | Definição |
| ------ | ------ |
| + | Soma |
| - | Subtração |
| * | Multiplicação |
| / | Divisão |
| % | Resto |
| ++ | Incremento + 1|
| -- | Decremento -1 |

## Operadores de Comparação
| Operador | Definição |
| ------ | ------ |
| == | Igual |
| === | Identidade (Valor e tipo de dados igual) |
| != | Diferente |
| !== | Identidade Diferente (Valor e tipo de dados diferente) |
| > | Maior |
| < | Menor |
| >= | Maior ou igual |
| <= | Menor ou igual |

## Operadores de Lógicos
| Operador | Definição |
| ------ | ------ |
| && | AND (e) |
| \|\| | OR (ou) |
| ! | NOT (não) |

## Operadores de Atribuição
| Operador | Definição |
| ------ | ------ |
| += | Soma |
| -= | Subtração |
| *= | Multiplicação |
| /= | Divisão |
| %= | Resto |
